ANIMOA V1.5.3 — LANCEMENT
============================

1. Aucune installation n'est nécessaire.
2. Place le dossier Animoa dans C:\Dev.
3. Double-clique sur C:\Dev\Animoa\OUVRIR_ANIMOA.bat.
4. Si Windows bloque le fichier .bat, ouvre directement index.html.
5. Utilise toujours le même navigateur pour retrouver tes données.
6. L'application démarre désormais sans animal exemple.
7. Une mise à jour des fichiers conserve normalement les autres compagnons déjà enregistrés.
