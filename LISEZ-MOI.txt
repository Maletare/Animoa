ANIMOA 2.4.4 — VERSION NETTOYÉE

Cette archive contient le projet complet, sans le dossier interne .git.
Vous pouvez donc copier son contenu dans C:\Dev\Animoa sans écraser la liaison GitHub.

MISE EN LIGNE
1. Copiez tout le contenu de ce dossier dans C:\Dev\Animoa.
2. Acceptez le remplacement des fichiers existants.
3. Dans GitHub Desktop, vérifiez les changements.
4. Créez le commit puis cliquez sur Push origin.

SUPABASE
Les scripts d’installation et de maintenance sont rangés dans :
  supabase/sql/

Les modèles d’e-mails sont rangés dans :
  supabase/email-templates/

IMPORTANT
- supabase-config.js contient uniquement la configuration publique du site.
- Ne placez jamais une clé service_role, une clé Brevo ou un mot de passe dans les fichiers publics.
- La correction d’autorisation nécessaire aux rappels e-mail est intégrée aux scripts SQL.
